# 🚀 CPA Calculator

A vibrant, fully responsive web application designed to compute Cost Per Acquisition (CPA) instantly. Built with modern UI/UX principles, this tool helps marketers, business owners, and digital strategists optimize their ad campaigns with precision.

## 🎯 What does this project do?

The **CPA Calculator** takes the guesswork out of digital marketing analytics. By simply inputting total ad spend and the number of acquisitions, users can instantly calculate their Cost Per Acquisition (CPA). The application supports multiple global currencies, featuring real-time input validation and a premium, animated interface that makes data entry an engaging experience rather than a chore.

## ✨ Key Features

- **Instant Calculation:** Real-time computation on form submission.
- **Multi-Currency Support:** Quickly switch between popular global currencies (INR, USD, EUR, GBP, JPY, CAD, etc.) grouped systematically.
- **Premium UI/UX:** Features a dynamic animated gradient background and modern UI elements.
- **Fully Responsive:** Perfectly adapts to mobile, tablet, and desktop screens.
- **Accessibility & Validation:** Built with HTML5 validation and clear spacing to ensure a seamless interaction for all users.

## 💻 Technical Stack

- **HTML5:** Semantic architecture and accessible forms.
- **CSS3:** Advanced animations (`@keyframes`), Flexbox layouts, custom properties, and a highly polished digital aesthetic.
- **JavaScript (Vanilla):** DOM manipulation, event handling, and calculation logic without relying on heavy frontend frameworks.

## 🚀 How to Run Locally

You can run this project effortlessly in seconds—no build tools or dependencies required!

*(Note: Since this is a Vanilla HTML/CSS/JS project, no heavy dependencies are required. You can simply open `CPA -1.html` in your browser, or run a local dev server like `npx serve` inside the folder!)*

About Me

This project focuses on simplifying a core challenge in digital decision-making — understanding acquisition efficiency without friction. Instead of treating calculations as a backend task, the goal was to create a smooth, intuitive experience where users can instantly interpret their data.

The CPA Calculator is designed to be lightweight, responsive, and accessible, ensuring that users can quickly move from input to insight without unnecessary complexity. Special attention was given to interaction design, real-time validation, and visual feedback, making the process both efficient and engaging.

Beyond functionality, the project emphasizes clarity, usability, and adaptability across devices. It reflects an approach where even simple tools are built with precision, scalability in mind, and a focus on delivering real value through thoughtful design.
